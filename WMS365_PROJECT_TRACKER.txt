CURRENT SYSTEM SUMMARY
- Warehouse-side web app with desktop and mobile workflows.
- Shared server-backed data stored in PostgreSQL on Railway.
- Separate warehouse login and customer portal login.
- Company-scoped inventory so warehouse activity can be filtered to one 3PL customer at a time.

CURRENT FEATURE SET

1. Warehouse Operations
- Receive inventory by company, location, SKU, UPC, quantity, and tracking level.
- Track inventory by Units, Cases, and Pallets.
- Search by SKU, UPC, location, or multiple item list.
- Remove quantity, delete line, transfer stock, BIN-to-BIN move.
- Convert item workflow to move inventory from one item to another:
  Example: CASE to EACH, EACH to CASE, or re-SKU conversion using case-pack math.

2. Master Data
- Company master records.
- BIN / location master records.
- Item master records with:
  - SKU
  - UPC
  - Description
  - Tracking level
  - Units per case
  - Each dimensions
  - Case dimensions
  - Optional image
- Portal user setup by company with support for multiple users per company.

3. Labels
- Location label generation.
- Pallet label generation with:
  - Company
  - SKU
  - Description
  - Cases on pallet
  - Date
- Pallet lookup / reprint flow by pallet ID.

4. Reports
- Inventory by company.
- Inventory by location.
- Inventory by item.
- Excel-style inventory by vendor/company report.
- Pallet location and utilization reporting.

5. Billing
- Billing fee schedule by company.
- Manual billing line entry.
- Storage billing generation.
- Billing export for invoice processing.
- Auto billing hooks for receiving and shipped portal orders.

6. Customer Portal
- Portal login by company user email/password.
- Vendor/customer-scoped inventory visibility.
- Order entry with ship-to details and order lines.
- Draft and release order flow.
- Inbound request flow.
- Portal users only see data for their assigned company.

7. Shipping Confirmation
- Warehouse can move portal orders through:
  - RELEASED
  - PICKED
  - STAGED
  - SHIPPED
- Warehouse can upload shipped documents:
  - POD
  - signed BOL
  - packing slip
  - carrier confirmation
- Portal users can view shipped confirmation details and download uploaded documents.
- Shipment confirmation email support is now wired in for shipped orders.
