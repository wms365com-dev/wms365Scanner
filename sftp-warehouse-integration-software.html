SFTP FAQ

Questions file-driven customers usually ask.

Can one SFTP integration touch another customer's data?

No. WMS365 ties the integration lane to the saved company so import and export activity stays scoped correctly.

Can orders and inbounds use different folders?

Yes. WMS365 is set up to support separate folder paths for different SFTP workflows.

Can confirmations be pushed back out automatically?

Yes. Shipment confirmations, receipt confirmations, and inventory snapshots can be handled as outbound SFTP lanes.
