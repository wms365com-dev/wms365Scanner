Pure Foods Store Display Work Order Guide

This guide shows how to request a warehouse work order to pull multiple existing SKUs and build them into one finished store display, bundle, or kit SKU.

Pure Foods access status

Enabled for Pure Foods by Estee

  • Customer Portal is enabled.
  • Order Entry permission is enabled.
  • Added Services / Display Build Work Order is available in the portal.

Important inventory rule

The warehouse can only complete the work if every component SKU is available in inventory.

Before submitting, confirm all display components are on hand. If any SKU is short, receive or replenish the stock first.

Before You Start

Information needed for a display build

1. Component SKUsEach SKU that goes into the finished display, normally 4-6 SKUs.
2. Qty per displayHow many cases or units of each component are needed for one display.
3. Finished display SKUThe SKU that should be created after the display is built.
4. Displays to buildTotal number of finished displays requested.
5. ReferenceOptional sales order, purchase order, promotion, or project name.
6. Needed byDate the display build is requested, if there is a deadline.
Pure Foods stock is normally managed in cases. If the item should be managed as units, use the unit SKU ending in "s" where applicable.
Step preparationWMS365
Step 1

Open the Customer Portal

Go to www.wms365.co and sign in to the customer portal using your Pure Foods portal login.

After login, use the main menu to select Added Services.

  • Use Added Services for special warehouse work.
  • Use standard Sales Orders and Purchase Orders for normal inbound/outbound work.
Actual system screen - Added Services
Step 1 - Added ServicesPure Foods by Estee
Step 2

Select New Work Order

Actual system screen - New Work Order

Click New Work Order.

This opens the work order form for creating a display or bundle from multiple component SKUs.

  • Use it when 4-6 SKUs need to be pulled and packed into one finished display SKU.
  • Use clear quantities and notes so the warehouse can complete the build without calling for clarification.
Step 2 - New requestWMS365
Step 3

Screen 1: Enter the display components

Start by entering each SKU that goes into one finished display.

  • Component SKU: item the warehouse will pull from stock.
  • Qty Per Display: how many of that SKU go into one display.
  • Placement / Notes: optional shelf, layer, flavor, or packing note.
  • Use Add Component SKU if the display has more than four SKUs.
Check Inventory first. Every component SKU must be available before submitting the work order.
Actual system screen - Component SKUs
Step 3 - Componentssupport@wms365.co
Step 4

Screen 2: Enter the finished display item

After components are entered, fill in the item that should be created.

  • Finished Display SKU: SKU that should exist after the build.
  • Displays to Build: number of finished displays requested.
  • Needed By: requested completion date, if applicable.
  • References: optional sales order, PO, or promotion name.
Make sure the finished display SKU exists in Items first. If it is brand new, create it before submitting the work order.
Actual system screen - Finished display item
Step 4 - Finished displayWMS365
Step 5

Review the build summary and submit

Actual system screen - Build summary

Before submitting, review the summary and confirm the component totals make sense.

  • The system shows the quantity needed per display.
  • The system shows the total quantity needed for the full build.
  • If a component is short, do not submit yet. Replenish or receive stock first.
Submitting the request does not immediately change inventory. The warehouse reviews the request, performs the physical work, and then updates inventory.
Step 5 - SubmitWMS365
Step 6

Track the request status

After submission, open My Display Build / Kitting Requests to track status.

  • Submitted: warehouse has received the request.
  • In Progress: warehouse is working on it.
  • Completed: work is done and inventory should be updated.
  • Cancelled: request will not be completed.

Final Checklist

Before sending a work order to the warehouse

Customer checklist

  • All component SKUs are correct.
  • All component quantities per display are correct.
  • All component stock is available on hand.
  • Finished display SKU is correct and already set up.
  • Displays to build is correct.

What happens next

  • The request is sent to the warehouse team.
  • The warehouse pulls the components.
  • The warehouse completes the physical display build.
  • Inventory is updated after the work is completed.
Questions or changes should be submitted through WMS365 or sent to the warehouse email. This keeps the request history and inventory trail clear.
