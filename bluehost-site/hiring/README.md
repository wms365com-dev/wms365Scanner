# WMS365 Hiring Package

This folder contains public and sendable material for recruiting WMS365 sales members.

## Files

- `index.html`: public hiring page.
- `sales-member-role.md`: role details and commission rules.
- `recruiting-email.md`: email script to recruit potential sales members.
- `screenshots/01-hiring-sales-member.png`: ready-to-send screenshot of the public sales-member page.

## Suggested URL

Publish the page at:

`https://wms365.co/hiring`

## Current sales-member compensation

Sales members earn 25% recurring monthly commission on WMS365 software subscription fees actually collected from customers they bring in. Payout target is within 15 days after WMS365 receives customer payment.
