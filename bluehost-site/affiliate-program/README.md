# WMS365 Affiliate Program Sendable Package

This folder contains the public affiliate program page and the written sales pack you can send to affiliates or sales partners.

## Files

- `index.html`: public affiliate program page.
- `promotion-plan.md`: detailed partner plan, targets, tracking, and rules.
- `email-scripts.md`: ready-to-send affiliate recruiting and prospecting emails.
- `screenshot-shot-list.md`: list of screenshots to capture and send with outreach.
- `screenshots/`: ready-to-send homepage, pricing, and affiliate program screenshots.

## Suggested URL

Publish the page at:

`https://wms365.co/affiliate-program`

## Current offer

Affiliates earn 25% of monthly WMS365 software subscription fees collected from referred paying customers. Payout is within 15 days after WMS365 receives customer payment.
