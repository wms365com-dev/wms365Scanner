Traceability FAQ

Questions controlled-inventory teams usually ask.

Do all items need lot and expiration control?

No. WMS365 supports optional controls so only the items that need lot or expiration tracking carry that extra workflow.

Can the same SKU exist in multiple lots?

Yes. WMS365 is designed to support the same SKU across multiple lot numbers and expiration dates.

Can pick priority follow earliest expiration?

Yes. FEFO-style allocation can reserve the stock with the sooner expiration so the right inventory is picked first.
